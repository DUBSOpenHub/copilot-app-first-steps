# Brand guidance

This project follows the public [GitHub Copilot Brand Toolkit](https://brand.github.com/brand-identity/copilot).

Official asset used:

- `assets/brand/github-copilot-product-lockup.png`
- Source: `https://brand.github.com/_next/static/media/copilot-01.0e1505e6.png`
- Usage: included unmodified as a product-lockup reference, not as the project logo.

## Applied guidance

- Use mostly neutral GitHub surfaces.
- Use Copilot Purple `#8534F3` as the primary Copilot accent.
- Use Copilot Orange `#F08A3A` and `#FE4C25` sparingly for warmth and momentum.
- Keep GitHub Green usage minimal and functional.
- Do not hero the deprecated standalone Copilot logo.
- Do not recreate or modify GitHub logos or product lockups.
- Do not imply GitHub endorsement.
- Use mascot-style visual energy sparingly; the page currently avoids mascot artwork entirely.

## Design intent

The page is designed from the learner's perspective:

- What can I create?
- Why is this valuable to me?
- How do I stay in control?
- How do I become a builder, even if I am not technical yet?

## Current implementation

- `index.html` uses the official Copilot palette values.
- `assets/hero.svg` uses neutral surfaces plus Copilot purple/orange accents.
- The README includes the official Copilot product lockup reference unmodified and avoids using a GitHub logo as the project logo.
